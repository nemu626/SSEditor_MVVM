# SSEditor_MVVM
会話文特化型エディタ製作中。C#,WPF

WPF,MVVMの勉強として鋭意製作中です。

----
Ctrl, Alt + 0~9に人物選択のホットキー割り当て可能。
Alt + Enterで文章の入力完了。
Ctrl + Enterで地の文入力
